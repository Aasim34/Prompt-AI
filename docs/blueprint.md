# **App Name**: PromptForge AI

## Core Features:

- Idea Input: A textbox where users can input their idea.
- Output Goal Selection: A dropdown menu allowing users to select the desired output goal (e.g., Website Prompt, App Prompt).
- Prompt Generation: Generates a detailed, formatted prompt based on the user's idea and selected output goal.
- Clipboard Access: Functionality to copy the generated prompt to the clipboard, or download the text file.
- Example Prompts: Show examples of user prompts to help understand potential of app.
- Enhanced Prompt Refinement: Utilize an AI model as a tool to auto-enhance and refine the generated prompt for clarity and effectiveness, if requested by the user.
- User Saved Prompts: Allows users to log in and save their past prompts for future use.

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) to convey expertise.
- Background color: Off-white (#F9FAFB) for a clean, modern feel.
- Accent color: Soft lavender (#818CF8) for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean, objective style.
- Clean line icons for intuitive navigation.
- Minimalist layout with a focus on readability and ease of use.
- Subtle glow effect for the generated prompt box to highlight the key output.
